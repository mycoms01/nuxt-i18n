# Setup

Add nuxt-i18n to your dependencies using Yarn:

yarn add nuxt-i18n

Or NPM:

npm i nuxt-i18n

Then add the module to nuxt.config.js:

{
  modules: [
    ['nuxt-i18n', {
      // Options
    }]
  ]
}
